常用插件

Android

  • ButterKnifeZelezny 5 星
  • ADB Comand 3星
  • GsonFormatString 解析成实体 Default Option + s(Mac), Alt + s (win)
  • Key Promoter X 快捷键提示 AS plugin 搜索安装
  • codota 代码案例查找【重点】 快捷键 Shift + Alt + Space
  • CodeGlance 代码概况
  • Parcelable Plugin
  • JarFilterPlugin 修改第三方Jar并打包进应用，如可以借助 JarFilterPlugin 来修改 Retrofit 的内部代码

Java

自动生成 serialVersionUID

  • lombok

bashsupport

  • IDEA中编写脚本并运行shell脚本

优化

  • dexcount-gradle-plugin 每次编译都统计Field and Method number

UI

  • android-drawable-importer
  • android-selector
  • Material Design Icon Generator
  • Android-drawable-preview-plugin
  • Android Styler 相对于AS提供的功能，这个优势在能指定style存放的文件及相对的位置；

其他

  • idea-multimarkdown【收费】
  • gradle-maven-publish-plugin 上传你的代码到所有的版本仓库中
  • DepSwitchPlugin aar和源码切换，开发阶段源码，稳定阶段用aar

Android / Intellj 共用

1. Translation

用法详解

  1. Translate and Replace 【重点应用】用于方法和类的命名上提高代码的可读性

  2. Translate Documentation 能翻译文档并且自动排版，非常易于阅读

  3. Mac Pro 快捷键

    1. Ctrl + command + i 快速翻译框
    2. +U 翻译 / + O translter replate

2. codota

【注意】：通过查找关联的案例，提高自己的编程能力

使用方式

  1. 选中要查询的类->
    1. 右击 get relevant examples / (Mac Pro )ctrl + Shift + O
    2. window Ctrl + Shift + O

3. Rainbow brackets 不同层级的brackets显示不同颜色

Lombok 自动生成 get、set toString

Android Studio

  1. 安装 Lombok Plugin
  2. 增加依赖

 compileOnly 'org.projectlombok:lombok:1.18.10'
 annotationProcessor 'org.projectlombok:lombok:1.18.10'
}
